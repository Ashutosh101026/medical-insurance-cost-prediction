# Medical Insurance Cost Prediction

## Overview
This project predicts medical insurance costs using Linear Regression and Feature Engineering techniques.

## Dataset Features
- Age
- Gender
- BMI
- Number of Children
- Smoking Status
- Region

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Workflow
1. Data Preprocessing
2. Exploratory Data Analysis
3. Feature Engineering
4. Linear Regression Modeling
5. Model Evaluation

## Results
Best R² Score Achieved: 0.865
